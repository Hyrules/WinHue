Features

Groups

Manage light groups on the bridge, to categorise your lights or just group together multiple lights from one fitting.

You can also create and edit the rooms that are shown in the official Hue app.

Rules

Edit rules on the bridge in detail, with an easy to use interface for selecting the rule's parameters. You can even load and edit existing rules without having to start from scratch.

Floorplan

Create and edit floorplans that match your house. You can import your own images to set up a floorplan that works for you.
